Zignis

沉淀最佳实践，减少重复投入，统一实现方式。

快速上手 →

一致

不管Node项目使用何种框架，或者如何抽象分层，都可以使用本框架来实现统一风格的命令行脚本。

可扩展

插件可以扩展，命令可以覆写，配置可以覆盖，使用钩子机制，可以与内置或第三方插件定义的钩子交互。

高效

因为规则简单，所以开发效率高，因为使用频繁，所以工作效率高。

# 对于本地环境，一般推荐全局安装
npm i -g zignis
zignis help
# 首次在项目中整合
cd YOUR_PROJECT
npm install zignis
zignis init
zignis make command test
zignis make command application/test --extend=zignis
